Политика конфиденциальности

Этот инструмент — личное приложение, предназначенное для загрузки собственного видеоконтента владельца в черновики (inbox) его же TikTok-аккаунта через официальный TikTok Content Posting API.

Какие данные используются

  • Access-токен и refresh-токен, полученные через OAuth 2.0 от TikTok, для авторизованного доступа к аккаунту пользователя.
  • Видеофайлы, которые пользователь сам загружает через интерфейс приложения, — временно, только на время передачи в TikTok.

Как используются данные

Токены используются исключительно для вызова методов TikTok Content Posting API от имени владельца аккаунта. Видеофайлы передаются напрямую в TikTok и удаляются с сервера приложения сразу после отправки.

Хранение и передача третьим лицам

Данные не передаются третьим лицам. Токены хранятся локально на сервере, который контролирует владелец приложения. Приложение не собирает аналитику и не использует данные в рекламных целях.

Удаление данных

Пользователь может отозвать доступ приложения в любой момент через настройки своего TikTok-аккаунта (Settings → Security → Manage apps), после чего токены становятся недействительными.

Контакт

По вопросам, связанным с этим приложением, обращайтесь к его владельцу напрямую.
