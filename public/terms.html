Условия использования

Это личное приложение для загрузки собственного видеоконтента владельца в черновики (inbox) его же TikTok-аккаунта через официальный TikTok Content Posting API.

Назначение

Приложение предназначено для одного пользователя — его владельца. Оно не предоставляется как публичный сервис третьим лицам и не монетизируется.

Ответственность пользователя

  • Пользователь самостоятельно отвечает за содержание загружаемых видео и его соответствие правилам сообщества TikTok.
  • Приложение не публикует контент автоматически — каждое видео требует ручного подтверждения публикации внутри приложения TikTok.
  • Пользователь обязан соблюдать условия использования TikTok и TikTok for Developers при работе с этим приложением.

Ограничение ответственности

Приложение предоставляется "как есть", без каких-либо гарантий. Владелец приложения не несёт ответственности за действия TikTok, включая ограничение или блокировку аккаунта, произошедшие по причинам, не зависящим от работы этого приложения.

Изменения

Владелец приложения может изменять эти условия и функциональность приложения в любое время.

Контакт

По вопросам, связанным с этим приложением, обращайтесь к его владельцу напрямую.
